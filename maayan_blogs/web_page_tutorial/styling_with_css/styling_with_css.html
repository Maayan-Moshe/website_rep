Styling with css

  1. Open notepad++ and start a new file by clicking File→New.
  2. Save the file in the same folder as your *.html file by the name "home_page_style.css".
    You can do it by clicking File→Save. Now navigate the the relevant folder in the "Save in:" text box. Choose "Cascade Style Sheets File (*.css)" in the "Save as type:" dropbox. Write the name of the file and save.
  3. Add the following line to "home_page_style.css":
    html { color: rgb(40,40,40); background-color: rgb(220,220,220); font-family: arial, Sans-serif;}
    This will change the color of the fonts to a little bit lighter than black. The background color to a little bit darker than white. The two things should make it easier for the user to read the content without trying his eye. Also it will change the default font to Arial.
  4. Add the following line to your *.html file:
    <link rel="stylesheet" href="home_page_style.css">
    and look on the result on your browser.
  5. Add h1, h2, h3{ background-color: SteelBlue; color: white;} to make titles blue with white text.
  6. Add h1, h2 { text-align: center;} to make heavy titles centred text.
  7. Add to the *.css file the lines:
    header, article, footer{ width: 1000px; background-color: rgb(245,245,245); padding: 10px;}
    .center { margin: auto; }
  8. Change to the line <header> to <header class = "center">. This will make the header in the center of the page.
    Do the same thing the the <article> and <footer>.
