Adding content to your own web page using only html5

  1. Open maayan_moshe_home_page.html with notepad++.
  2. Add a line under <h2>About me</h2> and write <p></p>. <p></p> stands for paragraph.
  3. Add your content as text between <p> and </p> in my case the paragraph looks like this:
    <p>There is nothing better than finding original, simple and unique solution to a new problem. Learning new fields and finding out of the box solutions is what I do.I am mathematician, physicist and software developer by this order. I am enthusiastic about applying mathematics to real life problems relating seemingly non-connected fields.</p>
  4. Adding a definition list, basically list with titles. Add Under <h3>Education</h3> add a new line.
    1. Add a new element <dl></dl>.
    2. Add the line
      <dt><strong>PhD</strong>, <i>Condensed Matter Physics (theory) - Tel Aviv University</i>, 2005-2009</dt>.
      <dt> stands for new definition. <i> stands for italic text format.
    3. Under the above line add the lines
      <dd><p>During this period, I published several papers in peer reviewed journals (5 papers in Physical Review B, 2 papers in Journal of Physics, and 3 conference presentations) and won prizes and scholarships.</p> <p><i>Activities and Societies</i>: Research and Theaching assistant</p> </dd>
    4. Add the rest of the education content in similar manner from here.
  5. Add a new definition list under <h3>Experience</h3>. You can get the content from here.
  6. Adding unordered list. In the forth item in the experience list we add unordered list.
    1. Add a new element <ul></ul>.
    2. Inside the new element add:
      <li>Derivative pricing methods using Merton model to identify the best features for new debt issuance.</li>
    3. Add the other bullet content in a similar manner from here.
  7. Add footer content under <h2>Contact details</h2>. You can get the content from here.
