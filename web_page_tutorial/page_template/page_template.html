The first steps to make your own web page template using only html5

  1. Install notepad++ from here. Just click on the Download button and follow instructions (basically click next all the time).
  2. Open notepad++ and start a new file by clicking on file→new.
  3. First we write a general template for html5 page. Write the following lines:
    <!DOCTYPE html>
    <html lang="en">
    <head>
    <meta charset="utf-8">
    <title>page template example - Maayan Moshe's home page</title>
    <meta name="description" content="Maayan Moshe's home page">
    <meta name="keywords" content="Maayan Moshe, curriculum vitae, home page">
    </head>
    <body>

    </body>
    </html>
  4. Next we introduce the sections from which the page will be composed of. Inside the body section write:
    <header>

    </header>
    <article>

    </article>
    <footer>

    </footer>
  5. Adding the header content to the home page. Inside the <header> section add:
    <h1>Maayan Moshe's home page</h1>
  6. Adding the article content head lines. Inside the <article> section add:
    <h2>About me</h2>

    <h2>Curriculum vitae - Maayan Moshe Ph.D.</h2>

    <h3>Education</h3>

    <h3>Experience</h3>
  7. Adding the footer content head lines. Inside the <footer> section add:
    <h2>Contact details</h2>
  8. Save the the resulting file.
    1. Click file→Save As.
    2. In the "Save as type:" option box choose "Hyper Text Markup Language file (*.html;*.htm;*.shtmal;*.shtm;*.xhtml;*.hta)".
    3. In the "Save in:" text box choose your desired folder. (If you are using git reposetory please save it in the reposetory folder.)
    4. In the "File name:" test box write "maayan_moshe_home_page".
    5. Click the "Save" button.

    You can view the result of this page instructions here. You can view your file by double clicking it and opening it with the browser.
